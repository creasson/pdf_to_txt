# SimCLR-Tensorflow

Unofficial Tensorflow Implementation of "A Simple Framework for Contrastive Learning of Visual Representations"

### Requirements:

- pip install Pillow

- pip install -U scikit-learn scipy matplotlib

- pip install tensorflow 2.0

### Training and testing

- python main.py
